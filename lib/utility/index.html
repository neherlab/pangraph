PanGraph.Graphs.Utility.cost — Constant
cost = (
	open   = -6.0,
	extend = -1.0,
	band   = (
		lower = Inf,
		upper = Inf,
	),
	gap    = k -> k == 0 ? 0 : cost.open + cost.extend*(k-1),
	match  = (c₁, c₂) -> 6.0*(c₁ == c₂) - 3.0,
)

cost are the default dynamic alignment parameters used.
